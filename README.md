# Sudoku Solver

My first Rust project. The goal was to get to understand the syntax and basic concepts of
Rust language.

## Requirements

* Tested with Rust 1.37.0

## Build and run

Clone the repo
```
git clone https://github.com/Miksu82/sudoku-solver
```

Run
```
$ cargo run
```

Run tests
```
$ cargo test
```
